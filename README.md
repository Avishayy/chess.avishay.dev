# chess.avishay.dev

Landing page for [Chess Position Explorer](https://chess.avishay.dev), a ChatGPT app that adds an
interactive chessboard to ChatGPT conversations.

Static site served via GitHub Pages. Custom domain configured via `CNAME`.

## Structure

- `index.html` — landing page
- `privacy/` — privacy policy
- `terms/` — terms of service
- `demo.mp4` — demo video
- `logo.png` — app icon (also used in the hero)

## Deployment

Push to `main`. GitHub Pages serves from the repo root.
